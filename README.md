# Synsepalum Dulcificum

## Taste ye the Miracle Fruit!

Dulcificum changes the ~flavor~, or dialect, of 3d printer commmands.

Supported dialects include:
+ MiracleGrue jsontoolpath dialect
+ UltiMaker GCode dialect

## build

```angular2html
mkdir build
cd build
conan install ..
conan build ..
```
